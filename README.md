# Snake
1v1 Snake Game
launch with "python snake.py [game mode] [difficulty] [theme] [music volume]"
[] arguments are optionnal
------------------------------
------------------------------
Game Modes : 
 --> infinity : Wrap around edges
 --> time : Become as long as possible in 60 seconds
 --> god : Be invincible 
------------------------------
Game Difficulty : 
 --> 1 : Normal
 --> 2 : Hard
 --> 3 : Good Luck
 (decimals are accepted)
------------------------------
Game Themes : 
 --> classic
 --> epic
 --> gfunk
 --> retro
 --> metal
 --> chill
------------------------------
------------------------------
